# 实验平台与实验环境说明

本书所有配套实验默认采用如下两个免费的模拟器：      
1）QEMU   
2）香山模拟器NEMU     

我们推荐使用我们提供的VMware/VirtualBox镜像，下载方式：关注“奔跑吧Linux社区”微信公众号，输入“risc-v”获取下载地址。   
配置如下：
1. 主机硬件平台：Intel x86_64处理器兼容主机。
2. 主机操作系统：Ubuntu Linux 20.04。
3. GCC版本：9.3（riscv64-linux-gnu-gcc）。
4. QEMU版本：4.2.1
